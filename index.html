Covid-19 Browser is a web application aimed at helping researchers browse a vast variety of research papers collected and distributed by Kaggle. The purpose of this project is to provide the non-data-scientists, a way to understand the documents. This web application will allow readers to:

  • Perform search within the document content by keywords
  • Document search using Title, Author Name, Journal, or Paper ID
  • Generate on-demand summarization of documents
